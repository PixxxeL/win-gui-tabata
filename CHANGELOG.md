# Changelog

Формат — по мотивам [Keep a Changelog](https://keepachangelog.com/ru/),
версии — [SemVer](https://semver.org/lang/ru/).

## [1.1.0] — 2026-07-23

### Добавлено

- Окно «Итоги» по «Стоп» или завершении сценария: время по каждому периоду
  (подготовка и паузы не учитываются) и общий итог. Формат времени —
  `45 сек` / `3 мин 20 сек` / `1 ч 2 мин 5 сек`.
- Выбор расписания кликом по названию (когда расписаний в конфиге больше
  одного и таймер в покое). Выбор действует до перезапуска приложения.
- Атрибут расписания `askProjectName`: при старте запрашивается название
  проекта (до 256 символов); оно добавляется тегом к именам периодов в логе,
  аналитике и итогах — «Работа [Проект №1]». Пустой ввод — без тега.
- Блур и затемнение содержимого главного окна под модальными окнами.
- Второе расписание «Простой 45/15» в конфиге по умолчанию.
- Лог аналитики `analytics.log` в каталоге данных: статус (ENABLED/DISABLED)
  и ответы сервера AppMetrica — ошибки отправки больше не глотаются молча.
- Сборка релизов в GitHub Actions по тегу `vX.Y.Z`: два zip-архива —
  self-contained (рантайм внутри) и no-runtime (нужен .NET 8 Desktop Runtime).
  Версия приложения подставляется из имени тега.

### Изменено

- Структура репозитория: код перенесён в `src/`, ассеты — в `assets/`.
- Сборка MSIX в CI временно отключена (до одобрения имени в Microsoft Store);
  шаги сохранены закомментированными в workflow.
