# Schedule Timer — таймер, который работает по вашему сценарию

[English version](README.md)

Универсальный таймер-расписание для Windows с циферблатом и перемоткой
по клику. Настраивайте вложенные сценарии произвольной сложности (помодоро,
табата и другие виды) — повторы, подготовку. Можно указывать свои звуки
на события.

Запускайте один таймер под разные проекты — по стопу получите время
по каждому периоду и итог. Без установки и аккаунтов: `exe` плюс один файл
конфига.

Для сфокусированной работы, тренировок и всего, что идёт по таймингу.
